# 🌐 FUTURE_FS_Task1 - Personal Portfolio Website

## 🚀 Project Overview
This is Task 1 of the Future Interns Full Stack Web Development Internship.

It is a Personal Portfolio Website built using HTML and CSS to showcase personal details, skills, and basic web development understanding.

- Responsive layout for all devices
- Hero section with introduction
- About section (personal details)

- Simple and beginner-friendly structure
